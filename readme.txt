Bilingual (English/Japanese) enrollment form as a Divi module, with reCAPTCHA v3, email delivery, and optional encrypted export.

== Description ==
Add the “Bahá’í Enrollment Form” module in the Divi Builder. Configure field texts in the module, and set email + reCAPTCHA keys under **Bahá’í Enrollment** settings.

== Changelog ==
= 1.1.0 =
* Fully integrated Divi module (no shortcode).
* Per-field text settings in module; Visual Builder support.
* Admin settings limited to email/reCAPTCHA + export.
